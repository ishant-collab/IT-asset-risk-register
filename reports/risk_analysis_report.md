# Risk Analysis Report

## Organization Overview
This project is based on a hypothetical mid-sized IT organization
with approximately 50 employees and standard IT infrastructure,
including servers, endpoints, networks, and data systems.

## Scope of Assessment
The risk assessment covers the following IT assets:
- Employee Laptops
- Web Server
- Email System
- Customer Database
- Wi-Fi Network

## Risk Assessment Summary
The risk analysis identified multiple cybersecurity risks
with varying impact and likelihood levels.

High-risk assets include:
- Web Server (SQL Injection risk)
- Customer Database (Data breach risk)

Medium-risk assets include:
- Employee Laptops
- Email System
- Wi-Fi Net
