# IT Asset Risk Register

## Project Overview
This project demonstrates a beginner-friendly IT Asset Risk Register
designed to identify, analyze, and mitigate cybersecurity risks
within an organization.

## Objectives
- Identify critical IT assets
- Analyze cybersecurity threats and vulnerabilities
- Calculate risk using impact and likelihood scoring
- Classify risks into Low, Medium, and High
- Recommend appropriate risk mitigation strategies

## Risk Scoring Model
Risk is calculated using the following formula:

Risk Score = Impact × Likelihood

| Score Range | Risk Level |
|------------|-----------|
| 1–6 | Low |
| 7–14 | Medium |
| 15–25 | High |
